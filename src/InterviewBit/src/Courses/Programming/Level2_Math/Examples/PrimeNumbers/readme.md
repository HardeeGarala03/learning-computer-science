# Prime Numbers

## Origin

[Link](https://www.interviewbit.com/problems/prime-numbers/)

## Description

Given a number N, find all prime numbers up to N ( N included ).

Example:

if N = 7,
all primes till 7 = {2, 3, 5, 7}

Make sure the returned array is sorted.
