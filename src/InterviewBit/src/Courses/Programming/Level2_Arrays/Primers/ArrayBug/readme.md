# Array Bug

## Origin

[Link](https://www.interviewbit.com/problems/arraybug/)

## Description

Rotate the array A by B positions.

So, for example,

```text
A : [1 2 3 4 5 6]
B : 1
```

The output

```text
[2 3 4 5 6 1]
```
